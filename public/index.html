Setup Guide
Backend URL

Current backend:

To point to a different backend, edit public/config.js

Step 1 — Webhook
GitHub → Source repo → Settings → Webhooks → Add webhook

URL:
https://your-backend.vercel.app/webhook

Content-type: application/json
Events: Push only
Step 2 — Docs Repo
Create a new empty GitHub repo named
QuantumLangCodeExplaination

under your account. The backend will auto-commit generated READMEs there.
Step 3 — Deploy
cd qlce-backend
npm install
vercel --prod cd qlce-frontend
npm install
vercel --prod
Then update config.js with backend URL.
